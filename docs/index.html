Backtest V3 — Top-Down HTF + Break-Even Management

Filtro H4→H1 para sesgo direccional · Entrada V1 (inducción + vela de fuerza) · SL a break-even al 1.5:1 · 2024–2025

Cómo funciona V3
H4 detecta la macro-estructura (barrido de swing o HH/HL vs LH/LL). H1 confirma. Si hay conflicto → neutral → no operar ese día. Solo se entra si la dirección del setup coincide con el sesgo H4/H1.
Break-Even automático
Cuando el trade va +1.5× el riesgo en ganancia, el SL se mueve al precio de entrada. Resultado "BE" = PnL $0, no cuenta como pérdida, permite buscar T2.
